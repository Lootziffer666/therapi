# THERAPI Spec Pack

Enthalten sind:
- 7 kanonische HTML-Screens des Prototyps
- `DESIGN.md` als verbindliche Design-Spezifikation
- `COMPONENT_OVERVIEW.md` als Komponentenbaukasten

Kanonische Screens:
1. control-panel.html
2. endpoint-explorer.html
3. schema-drift.html
4. capture-inspector.html
5. openapi-export.html
6. collections-export.html
7. settings.html

Wichtige Korrektur:
- Code-/Snippet-Darstellung ist **hell und neutral**, nicht dunkel.
- `schema-drift.html` gehört zum verbindlichen Screen-Set.
